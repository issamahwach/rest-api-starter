# Node.js REST API - Typescript

## Overview
Simple and structured code for building REST APIs with Node.js using Typescript.

## Technologies
- Node.js
- MongoDB with Mongoose
- Express.js
- Typescript
- Zod Validation

## Concepts
- Rest API principles (CRUD & HTTP)
- JWT & Refresh Token
- Request Validation
